A Demo for Data Analysis, Adaption of Collaborative Filtering & Content Based Filtering System

Amazon Product Recommendation

In this project, I developed an Amazon product recommendation system based on user-item interaction data using both collaborative and content-based filtering techniques. The goal was to enhance user experience by suggesting products tailored to individual preferences. Leveraging user-item interaction data and product attributes, I employed various algorithms to generate accurate and personalized recommendations. This project showcases my skills in data analysis, machine learning, and Python programming, with a focus on real-world applications in e-commerce.
