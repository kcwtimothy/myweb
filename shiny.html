A prototype Rshiny app for Data Management and Lineage

Rshiny app: Data Management & Lineage Tool

In many organizations, managing data effectively can be a complex and challenging task. This prototype was created to address some of the common pain points in data management, including:

  • Data Lineage Tracking: The app allows users to create and manage data lineage tables, providing a clear view of data transformations and dependencies.
  • Dynamic Table Creation: Users can create new tables on the fly, specifying column names and types.
  • Data Entry Management: The app provides interfaces for creating, updating, and deleting data entries across multiple tables.
  • Data Visualization: Utilizing the DiagrammeR package, the app offers visual representations of data relationships.
  • Dual Database Support: The prototype is designed to work with two separate SQLite databases, allowing for flexible data management scenarios.

This repository contains a prototype data management application that I developed during my time at my previous company. The application is built using R Shiny and is designed to streamline data management processes, improve data lineage tracking, and enhance overall data governance.
