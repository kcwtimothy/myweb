Amazon Product Recommendation II-Using LLM & XGB

This project aims to develop a robust product recommendation system for Amazon using advanced machine learning techniques. By leveraging a combination of Large Language Models (LLMs) for feature extraction and the powerful XGBoost algorithm for prediction, this system delivers personalized product recommendations to users.

  • LLM: Application of large language models (LLMs) for full-sentence word embedding & sentiment analysis
  • K-mean Clustering: Utilization of the K-mean clustering algorithm for feature extraction.
  • Modeling with XGBoost: Utilization of the XGBoost algorithm, known for its efficiency and high performance in machine learning tasks.
  • Hyperparameter Tuning: Optimization of model parameters using GridSearchCV and 4-fold cross-validation to achieve the best performance.
  • Assessment of model performance using Mean Squared Error (MSE) and R-squared Score (RÂ²).
